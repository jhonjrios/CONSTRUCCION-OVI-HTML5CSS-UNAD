OBJETIVO DEL CURSO

La idea principal de este curso es mostrar los fundamentos en lo que se refiere a la programacion web, comenzando por sus caracteristicas, requisitos para adentrarse en este mundo despues continuaremos en lo que se refiere a la Maquetacion que es el diseño previo de nuestra pagina web y de los distintos elementos que la contendran, seguiremos trabajando en lo que se refiere al lenguaje base que es HTML que no es mas que un lenguaje de marcacion y no de programacion, MySQL es una herramienta importante para el manejo de bases de datos y culminaremos con PHP, que nos brindara una manera mas dinamica de interactuar con la pagina y los usuarios que la visitaran

TEMATICAS A DESARROLLAR

  • Principios de Programacion
  • Maquetacion de una Pagina Web
  • Lenguaje HTML
  • Bases de Datos
  • Gestor de BD MySQL
  • Programando con PHP
